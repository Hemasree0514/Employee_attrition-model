# Employee_attrition-model
Predicting and analyzed employee attririon using ML.
This project predicts whether an employee will leave the company based on HR data.
I used Logistic Regression and performed data cleaning, feature engineering, and evaluation.

Tools Used:
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

Outcome:
Achieved 83% accuracy in predicting employee attrition.(leaving)
